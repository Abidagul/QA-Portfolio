# Process Flows

This folder contains business process flows, workflow diagrams, and system flow charts created for requirement analysis and QA testing activities.

## Included Documents

* External Organization Request Workflow
* Donation Module Process Flow
* Sponsorship Module Process Flow
* User Registration Workflow
* Business Process Diagrams

## Purpose

These flow charts help in understanding business workflows, approval processes, user journeys, and system behavior to support effective test planning and validation.

## QA Activities Supported

* Requirement Analysis
* Test Scenario Design
* Workflow Validation
* Business Process Verification
